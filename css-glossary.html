Standard Properties

Property Description Example values
background-color: Sets the background color of an element. brown; #1A5F7A; rgb(255,255, 128); hsl (50, 33%, 53%);
background-image Sets one or more background images on an element. url("...") url("...");
border-collapse Sets whether the cells inside a table have separate or shared borders. collapse; separate;
box-sizing: Sets how the total width and height of an element is calculated. 'content-box' gives you the default CSS box-sizing behaviour, whereas 'border-box' tells the browser to account for specified border/pading values. border-box; content-box;
clear: Sets whether an element must be moved below floating elements that precede it, but also applies to non-floating elements. none; left; right; both;
color: Sets the foreground color of an element's text and text-decorations. purple; #1A5F7A; rgb(255,255, 128); hsl (50, 33%, 53%);
content: Replaces content with a generated value, and can be used to define what is rendered inside an element/pseudo-element. url("..."); "/"; "emoji";
display: Sets whether an element is treated as a block or inline box, and the layout used for its children, such as a flow layout, grid or flex. block; inline-block; none; flex; grid;
empty-cells: Sets whether borders/backgrounds appear around table cells with no visible content. show; hide
float: Places an element to the left or right side of its container, allowing text and inline elements to wrap around it and removes it from the normal flow of the page. none; left; right
@font-face: Specifies a custom font, downloadedfroma remote server or locally installed, in which to dsplay text font-family: '...'; font-style: font-weight: 400; font-display: swap; src: url('...') format('woff2');
font-family: Specifies a prioritised list of one or more font-family and/or generic family names. 'Gill Sans', sans serif; cursive;
font-size: Sets the size of the font. 1.2rem; x-small; 12px; 80%;
font-style: Sets whether a font should be styled with a normal, italic or oblique face from its font-family. normal; italic; oblique;
font-weight: Sets the weight/boldness of the font. The availabilit depends on the font-family currently set. normal; bold; lighter; bolder; 85%
height: Specifies the height of an element/its content area. 150px; 6em; 75%; auto;
letter-spacing: Sets the horizontal spacing behaviour between text characters; positivevalues spread characters farther apart whereas negative values bring them closer together. normal; .2rem; -1px;
line-height: Sets the minimum height of a line box/to set the distance between the lines of text. normal; 2.5; 150%; 32px;
max-height: Sets the maximum height of an element to prevent the value of the height property from becoming larger than said height. 150px; 7em; 75%;
min-height: Sets the minimum height of an element to prevent the value of the height element from becoming smaller than said height. 150px; 7em; 75%;
opacity: Sets the opacity, which is the degree to which content behind an element is hidden. 0, 0.33%, 1;
position: Sets how an element is positioned in a document, used with the top, right, botto and left properties to determind the final location of positioned elements. static; relative; absolute; sticky; top: 40px;
text-align: Sets the horizontal alignment of the content inside a block element or table-cell. left; right; center; start; end; justify;
width: Sets the width of an elements content area, or the width of the border area when box-sizing is set to border-box. 150px; 75%; auto; 20em;
word-spacing: Sets the length of space between words and tags. normal; 4px; 1rem;
z-index: Sets the z-order of a positioned element and its descendants; overlapping elements with a higher z-index will cover those with a smaller one. auto; 1; -1;

Shorthand properties

CSS properties that let you set the values of multiple other CSS properties acting on the same theme simultaneously, allowing for more concise and readable style sheets.

Shorthand property Description Example values
border: Sets an elements border using the values of border-width, border-stye and border-color. 1px, solid, white;
border-radius Shorthand property that rounds the corner of an element's outer border edge, often used to make circular corners. 25% 10%; 10% 40% 50%; 10px 100px;
font: Shorthand property that sets all the different properties of an element's font, including font-style, font-weight, font-size, line height, and font-family. italic, bolder, 16px/1.5, 'Bebas Neue', sans serif;
margin: Shorthand property that sets the margin area (the space around an element's border) on all four sides (-top, -right, -bottom, -left) of an element. Can use -top, -right, -bottom or -left to specify only one side. 10px 50px 20px 0px; 0x; 1em; 5% 10%; auto;
overflow: Shorthand property that sets the desired behaviour when content does not fit in the parent element box horizontally (overflow-x) or vertically (overflow-y). visible; hidden; scroll; auto;
padding: Shorthand porperty that sets the padding area (the inner space surrounding the element) on all four sides of an element (-top, -right, -bottom, -left) at once 10% 0; 10px 25px; 10px 30px 50px 70px;
scroll-margin Shorthand porperty that sets all four sides (-top, -right, -bottom, -left) of an element's scroll margin at once. 20px; 0; 2em;
text-decoration: Shorthand property that sets the appearance of decorative lines on text. underline; underline dotted red; green wavy underline;

Pseudo-classes

A keyword added to a selector that specifies the state of the selected element.

Pseudo-class Description Example properties
:active Creates a pseudo-class that represents an element (e.g. a button or link) that is being activated by the user color: red; font-weight: bold; font-style: italic;
:hover Creates a pseudo-class for when the user interacts with an element with a cursor but doesn't necessarily activate it. color: red; font-weight: bold; font-style: italic;
link: Creates a pseudo-class that reperesents an element that has not yet been visited. color: red; font-weight: bold; font-style: italic;
visited: Creates a pseudo-class that applies to links that have been visited by the user. color: red; font-weight: bold; font-style: italic;

Pseudo-elements

A keyword added to a selector that lets you style a specific part of the selected element.

Pseudo-class Description Example properties
::after Creates a pseudo-element that is the last child of the seleced element, often used to add cosmetic-content (aka breadcrumbs) to an element with the content:"..."; property. content: ">"
::before; Creates a pseudo-element that is the first child of the selected element, often used to add cosmetic-content (aka breadcrumbs) to an element with the content:"..."; property content: "X";
